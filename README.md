# cpp-programs

A concise collection of standalone C++ programs organized by topic.

| Path | What you'll find |
| --- | --- |
| 01) Basic Programs | Basic C++ examples and simple exercises |
| 02) if-else and switch programs | Conditional statement examples and practice problems |
| 03) Loops | Loop-based problems |
| 04) Magic Number | Programs demonstrating "magic number" examples |
| 05) Pattern Questions | Pattern-printing problems and sample outputs |
